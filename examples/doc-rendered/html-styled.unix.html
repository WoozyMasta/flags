golden-doc

Golden doc parser

golden-doc [OPTIONS]

Description

Long description for golden tests. Includes options, groups and commands.

Options

Application Options

  • -v, --verbose - Enable verbose output
  • --config (default: config.yaml) - Path to config
    required environment MY_APP_APP_CONFIG
  • --mode (default: fast) - Execution mode
    choices fast, safe
  • --tag (default: api) - Tag filter
  • --header (default: x-env:dev) - HTTP headers
    kv :
  • --level [=info] - Log level
    optional info

Database Options

  • --db.host (default: 127.0.0.1) - Database host
    environment MY_APP_DB_HOST
  • --db.port (default: 5432) - Database port
    environment MY_APP_DB_PORT

Commands

run

Run command

Usage: golden-doc [OPTIONS] run [run-OPTIONS]

status

Show status

Usage: golden-doc [OPTIONS] status [status-OPTIONS]
